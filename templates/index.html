Tools Included

Emap2Sec

Identify secondary structures (α helices, β sheets, coils/turns) in a low to medium resolution cryo-EM map using deep learning.

MAINMAST

Build reliable candidate Cα models from a near-atomic resolution cryo-EM map without reference structures or manual intervention.

Emap2Sec+

Identify secondary structures and nucleic acids (RNA/DNA) in a low to medium resolution cryo-EM map using deep learning.

MAINMASTseg

Segment a near-atomic resolution cryo-EM map with symmetry constraints into several constituent density maps.
